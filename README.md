## Supported Models
Here are the following models Prompt Navigator works for:

* **Gemini** 
* **Claude**
* **ChatGPT** 
* **Copilot** 
* **Perplexity**
* **DeepSeek**
* **Poe**
* **Meta AI**
* **Grok**

## Troubleshooting
If you do not see the navigation buttons at the bottom right of the chat, please:
1. Refresh the page
2. Ask 1 prompt (if there aren't any) and refresh the page.

Most AI platforms (like ChatGPT or Gemini) use dynamic loading, so hitting refresh allows the Prompt Navigator to hook into the chat window correctly.
