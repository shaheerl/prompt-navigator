# Prompt Navigator
Chrome extension built to make prompt navigation within AI chats seamless. 

## Supported Models
Prompt Navigator is designed to work across the most popular AI interfaces:

* **Gemini** (Google)
* **Claude** (Anthropic)
* **ChatGPT** (OpenAI)
* **Copilot** (Microsoft)
* **Perplexity**
* **DeepSeek**
* **Poe**
* **Meta AI**
* **Grok** (xAI)

## 💡 Troubleshooting
If you do not see the navigation buttons at the bottom right of chat, please:
1. Refresh the page
2. Asking 1 prompt (if there aren't any already) and refresh the page.

Most AI platforms (like ChatGPT or Gemini) use dynamic loading, so hitting refresh allows the Prompt Navigator to hook into the chat window correctly.
